MoE TP Forward: Step-by-Step Example

Understanding Indices and Gates

The Scenario

  • Batch Size: 2 Tokens (T1, T2)
  • Top-k: 2 (Each token goes to 2 experts)
Indices Matrix (2x2)
Slot 0 (k=0) Slot 1 (k=1)
T1 (i=0) Expert 0 Expert 3
T2 (i=1) Expert 1 Expert 2
Gates Matrix (2x2)
Slot 0 (k=0) Slot 1 (k=1)
T1 (i=0) 0.7 0.3
T2 (i=1) 0.6 0.4

Correct Loop Structure

for k in range(self.topk): for i in range(batch_size): expert_idx = indices[i, k] gate_value = gates[i, k] # Expert takes a 2D array, so slice like x[i : i+1] expert_out = self.experts[expert_idx](x[i : i+1]) # Accumulate: gate * output # expert_out is (1, out_dim), so take [0] outputs[i] += gate_value * expert_out[0]
